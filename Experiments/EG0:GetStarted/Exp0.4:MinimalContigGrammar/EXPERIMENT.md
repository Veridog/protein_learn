# Exp0.4 — Contig 最小语法练习

## 目标

理解 RFdiffusion 的 contig 系统——这是所有条件生成任务的"语言"。通过最小实验探索 contig 参数如何影响生成的蛋白。

## 背景

Contig 是 RFdiffusion 的核心配置语言，决定哪些残基固定、哪些需要生成。格式：

```
contigmap.contigs=[<规格>]
```

### 基本语法

| 符号 | 含义 | 示例 |
|------|------|------|
| `N-M` | 生成 N 到 M 个随机长度残基 | `[50-100]` = 长度 50~100 的随机单体 |
| `N-N` | 精确生成 N 个残基 | `[100-100]` = 精确 100 残基 |
| `Ax-y` | 引用输入 PDB 的 A 链残基 x~y | `[A10-50]` = 引用现有结构 |
| `/0` | 分隔 target（固定）和 binder（生成） | `[A1-100/0 50-80]` = target A + 50~80 残基 binder |
| `L` | 单独的数字表示生成段落 | `[30/A10-50/20]` = 生成 30 → motif → 生成 20 |

## 任务

### 1. 长度扫描

固定所有参数，只改变生成的残基数量：

```
[50-50]  →  [100-100]  →  [150-150]  →  [200-200]  →  [300-300]
```

每个长度生成 20 个设计，回答：
- 长度变化对 pLDDT 的影响？
- 长度变化对结构多样性的影响？
- 多长时结构质量开始下降？

### 2. 固定 vs 随机长度

对比精确长度和随机长度：

```
[100-100]  vs  [50-150]  vs  [100-200]
```

- 随机长度是否产生更自然的蛋白？
- 最优长度（最低 scRMSD）是否集中在某个范围？

### 3. 长度与二级结构偏好

统计不同长度下 α-helix / β-sheet / loop 的比例分布：
- 短蛋白（< 100aa）倾向于什么二级结构？
- 长蛋白（> 200aa）的结构特征？
- 是否出现 RFdiffusion 论文中观察到的 "小蛋白偏好全螺旋" 现象？

### 4. Timesteps 敏感性

固定 contig `[100-100]`，改变扩散步数：

```
diffuser.T=20  →  50  →  100  →  200
```

- T 越小 = 更接近随机起点 = 结构更 diverse？
- T 和 scRMSD 的关系？
- 找到"性价比"最佳的 T 值

### 5. （可选）两个 Motif 段落

```
[30-50/A10-30/20-40]
```

用 Exp0.1 中评分最高的一个设计当作 "motif"，嵌入到新骨架中。这引入 motiff scaffolding 的最简形式。

```python

    {"contig": "[50-50]", "T": 50, "n": 20},
    {"contig": "[100-100]", "T": 50, "n": 20},
    {"contig": "[150-150]", "T": 50, "n": 20},
    {"contig": "[200-200]", "T": 50, "n": 20},
    {"contig": "[300-300]", "T": 50, "n": 20},
    {"contig": "[50-150]", "T": 50, "n": 20},
    {"contig": "[100-200]", "T": 50, "n": 20},
    {"contig": "[100-100]", "T": 20, "n": 20},
    {"contig": "[100-100]", "T": 100, "n": 20},

]
```
